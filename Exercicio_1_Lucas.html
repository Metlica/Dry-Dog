Html

No primeiro semestre aprendemos o basico em html como fazer

GIT

E na aula de hoje aprendemos a usar o GIT que é um sistema de
controle de versão distribuido e um sistema de gerenciamento de
codigo fonte, fizemos upload dos codigos , desenvolvemos codigos
passo a passo , tambm aprendemos a colaborar com outros usuarios
editando seus codigos.
